# Librarian bot

### Короткое описание
Телеграм-бот для систематизации сохранённых постов

### Какую проблему решает
В последнее время телеграм стал одним из основных каналов распространения информации.
К сожалению, функциональность "Избранного" (чат с самим собой для сохранения важных сообщений/постов) остаётся весьма ограниченной:
- Нельзя явно указать, к какой категории относится соответствующий пост
- Нельзя быстро найти все посты, относящиеся к какой-то категории

### Ценность
Данный бот позволяет структурировать пересланные ему посты подобно закладкам в браузере или файлам на компьютере: создавать папки произвольного уровня вложенности, перемещаться между ними, сохранять в них посты и, позднее, показывать все посты в папке.
